# Store React

## Setup:
1. Clone the repo.
2. Run `npm install`.
3. run `npm run start`.
4. For deploy use `npm run build`.
5. For test use `npm run test`. 
6. Go to http://localhost:3000 if the port is busy you need to read the console to know where is the app running.
